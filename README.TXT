Le projet est réaliser par : 
    python 3.8

Pour lancer le projet j'utilise la commande suivant : 
    py manage.py runserver 9090

Les packages utilisés dans ce projet:

    > Le package pandas
        pip install pandas
    
    > Le package numpy
        pip install numpy

    > Le package Scikit Learn
        pip install -U scikit-learn

    > Package qui nous permet de lier le projet avec une base donner NoSQL
        pip install djongo
    
    >Package permet d'uitiliser les commandes de MongoDB:
        pip install pymongo

    > Package de nettoyage les images dupliquer
        pip install django-cleanup

    > Package qui nous permet de crée une formulaire
        pip install django-crispy-forms
